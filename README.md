# 🛡️ VPN Manager (vpnmgr)

Интерактивная CLI-система управления VPN-протоколами (VLESS+XHTTP, Hysteria2, AmneziaWG, SOCKS5) для серверов на базе Debian 11.

## 🌟 Основные возможности

*   **🕹️ Интерактивный TUI**: Управление через удобные меню в терминале (whiptail).
*   **🔌 Множество протоколов**: Поддержка VLESS+XHTTP, Hysteria2, AmneziaWG и SOCKS5.
*   **👥 Управление пользователями**: Простой CRUD, лимиты трафика и статус подключений.
*   **📱 Легкое подключение**: Генерация URI и QR-кодов прямо в терминале.
*   **📊 Мониторинг**: Статус сервисов, графики трафика и логи в реальном времени.
*   **💾 Надежность**: Автоматические бэкапы, watchdog для перезапуска упавших сервисов.

## 🛠️ Технические требования

*   **ОС**: Debian 11 (рекомендуется)
*   **Оболочка**: Bash ≥ 5.0
*   **Зависимости**: `jq`, `whiptail`, `curl`, `qrencode`, `openssl`

## 🚀 Быстрый старт

```bash
# Скопируйте проект на сервер
# Запустите установщик
sudo bash install.sh
```

## 📂 Структура проекта

*   `vpnmgr.sh` — точка входа.
*   `lib/` — функциональные модули системы.
*   `templates/` — шаблоны конфигураций протоколов.
*   `data/` — базы данных (JSON).
